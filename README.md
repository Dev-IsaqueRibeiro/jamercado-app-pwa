🛒 JáMercado — Sua Lista de Compras Inteligente

Otimize sua rotina, economize tempo e nunca mais esqueça um item essencial.

O JáMercado é um Progressive Web App (PWA) desenvolvido para transformar a experiência de fazer compras.
Focado em simplicidade e agilidade, ele permite o gerenciamento de listas em tempo real, com suporte total para 
dispositivos móveis e funcionamento offline.


🚀 Diferenciais do Projeto

• Experiência PWA Nativa: Instalável em Android e iOS, com carregamento instantâneo e ícone na tela inicial.

• Sincronização em Tempo Real: Utiliza Firebase Firestore para garantir que suas listas estejam sempre atualizadas
em todos os seus dispositivos.

• Autenticação Segura: Login social via Google integrado com Firebase Auth, garantindo privacidade e facilidade de acesso.

• Foco em UX (User Experience): Interface limpa, intuitiva e otimizada para o uso com uma mão durante as compras no
supermercado.

• Suporte Offline: Graças ao Service Worker, você pode consultar sua lista mesmo em locais com sinal de internet instável.


🛠️ Stack Tecnológica

Este projeto foi construído utilizando as melhores práticas de desenvolvimento web moderno:

• Frontend: HTML5, CSS3 (Design Responsivo) e JavaScript (ES6+).

• Backend as a Service (BaaS): Firebase (Firestore, Authentication & Hosting).

• PWA: Web App Manifest e Service Workers para suporte offline e instalação.

• Domínio Customizado: Configuração de DNS e SSL via Firebase Hosting para o domínio jamercado.com.br.


📦 Funcionalidades Principais

Gestão de Itens: Adicione, edite e remova produtos da sua lista com um toque.

• Checklist Interativo: Marque os produtos conforme os coloca no carrinho.

• Sincronização Cloud: Seus dados são salvos na nuvem automaticamente.

• Política de Privacidade Integrada: Conformidade com boas práticas de transparência de dados.

🔧 Como Rodar o Projeto Localmente

1 - Clone este repositório:

Bash
git clone https://github.com/Dev-IsaqueRibeiro/jamercado-app-pwa.git

2 - Instale as dependências (se necessário):

Bash
npm install

3 - Configure seu arquivo lib/firebase.js com suas credenciais do Firebase.

4 - Rode através de um servidor local (como o Live Server do VS Code) ou via Firebase CLI:

Bash
firebase serve
